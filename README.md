# web-app-thing
Side project for working on a rewrite of Blockauth. Feel free to use as a boilerplate.

## what have you got
* Basic Bootstrap website
* JWT cookie based auth using Passport
* Discord OAuth

## how 2 use
* Node.js (v12)
* MongoDB
